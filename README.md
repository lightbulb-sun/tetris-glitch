# Tetris (GB) Line Clear Glitch Fix

## Changes

Fixes line clear glitch
in v1.1 of the Game Boy game
*Tetris*
in accordance with this
[romhacking.net forum thread](https://www.romhacking.net/forum/index.php?msg=449799)
by
[thirtythree](https://www.romhacking.net/forum/index.php?action=profile;u=104111).

## Download
The latest release can be found on the
[releases page](https://github.com/lightbulb-sun/tetris-glitch/releases).
Patch the original ROM with one of the `.bsdiff`, `.ips` or `.bps` files
to create the hack.
